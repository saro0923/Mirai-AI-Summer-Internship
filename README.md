## Objective

Upgrade the AI Multiverse Chatbot from a stateless chatbot to a stateful chatbot using Streamlit Session State.

## Features

✅ Streamlit Session State

✅ Chat History

✅ Persistent Conversation Memory

✅ st.chat_input()

✅ st.chat_message()

✅ Save User Messages

✅ Save AI Responses

✅ New Chat Option

✅ Saved Chat History

## Concepts Used

```python
st.session_state
```

```python
st.chat_input()
```

```python
st.chat_message()
```

The chatbot can now:
